BackEnd Side:

  • PHP 8.1
  • Composer 2.1.5
  • PHPUnit 9.5
  • cd TMS/backend
  • composer install
  • php .\vendor\phpunit\phpunit\phpunit --testdox tests

FrontEnd Side:

  • Angular 13.0.1
  • npm 8.1.2
  • npm install --global @angular/cli@latest && npm clear cache --force
  • cd TMS/frontend
  • change the baseUri in src/environments/environment.ts
  • npm install && ng serve
